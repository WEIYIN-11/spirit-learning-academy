# 靈獸學習學院

手機優先的學習小遊戲原型。玩家可以切換不同領域課程，閱讀短課程、完成小測驗，答對題目會獲得 XP，靈月靈狐會從幼年期一路成長到完全體。

## 目前功能

- 課程 catalog 架構，可載入不同領域
- Python 入門主題：變數、條件、迴圈、函式
- Git 基礎課程示範：版本控制與 commit
- 英文入門課程：句子結構、be 動詞、一般動詞、疑問句
- SQL 入門課程：資料表、SELECT、WHERE、ORDER BY / LIMIT
- 每課都有名詞解釋卡與圖解範例
- 每課 3 題選擇題
- 答題後會顯示「為什麼是這個答案」
- 答對得 XP，完成課程額外加成
- 使用整理後的靈月靈狐圖鑑呈現 10 個成長形態，保留每階名稱、靈獸樣貌與介紹
- 使用幼年期靈狐作為 PWA / 加入主畫面的 icon
- 靈獸互動：招手、搖尾巴、轉圈圈
- 靈獸可切換大版狀態卡或縮小成可拖曳陪伴浮窗
- `localStorage` 儲存進度
- 手機版單頁互動介面

## 使用方式

直接開啟 `index.html`，或在這個資料夾啟動本機伺服器：

```powershell
python -m http.server 5177
```

然後瀏覽 `http://localhost:5177`。
